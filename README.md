### Requirements

Node.js runtime for Windows, MAC, or Linux.

#### To install backend dependencies
```
$ cd backend
$ npm install
```

#### To install frontend dependencies
```
$ cd frontend
$ npm install
```

#### To run the App ( backend (Node.js and Express) and the frontend (using React.js) concurrently ) :- 
```
$ cd backend
$ npm run dev
```
